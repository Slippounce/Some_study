Этот проект представляет собой некий прототип отладчика. 
На вход подаём текстовый файл с кодом программы - можно создавать переменные, вызывать функции и т.д. 
в папке resources есть пример такого файла(template.txt) - программа начнёт использовать его, 
если не подать на вход другой файл аргументом.
после запуска можно управлять из консоли отладчиком, в начале работы выведется USAGE, где описан функционал отладчика
